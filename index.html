Sea Scene URP

Sea Scene URP is a 3D demo used to showcase Singapore Navy Ship. For this demo there will be 5 Navy Ships to be showcased. The Demo is made using Unity Engine with the URP project template. I made this project during my 1 year internship, and I learn a lot about making a reusable script, camera angles, Coroutine, some UI/UX tips and much more.

The main goal of this demo is to showcase the main feature for each ship, such as the ship's feature of landing aircrafts(if possible) and firing missiles/turret. In addition the other 3 features are, showing information about the ship, a view of the ship's control room and comparison of the ship's size with a human.

Mirror Multiplayer

Mirror Multiplayer is a prototype 3D multiplayer game where I make the base functions & features (such as Game Lobby, Character Controls, Mechanics, etc.) that will be used in a multiplayer game. During the development of this project I learn a lot about how to make a multiplayer game. The game was made using Unity and this is one of my project during my 1 year internship.

For this version I managed to create a Game Lobby where players can choose their server to create/connect room and players can also choose the role whether they want to be a server-only, a client or the host of the room. After that I made a feature for item interaction where the players can interact with object, such as pick up, drop and holding object. In addition to the feature I also experiment in spawning and deleting objects.

Oil Spill Response

Oil Spill Response is an immersive VR Simulation Game where the player will learn the steps on how to handle an Oil Spill on the ocean. The player will act as the employee that is responsible to prepare the tools needed for the operation. The player will also be guided on the tasks that must be perform.

The goal of the game is to complete all the tasks in the game which eventually leads to the process of handling the Oil Spill and removing it from the ocean. My role in this project is a Lead Programmer. In developing this project, I collaborated with 3D/2D artist for asset creation. The project was made for Shevia.

RPG Core Mechanic

This project was made to develop a core mechanic for an RPG game. VR 360 Video (Android)

VR 360 Video is a game where the main purpose is to simulate a VR 360 video in android platform. To simulate the VR experience, the control to look around is by using gyroscope of the android device, meaning the player can look around by moving around their android device. The Player can pause/start the video and select which video to watch from the thumbnail option or by cycling through the list using the next/prev button. In addition the player can also fast forward 5 second into the video or rewind to 5 second earlier. This project was made during my internship program.
Size: 287 MB
